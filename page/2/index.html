树莓派无显示器安装操作系统

发表于 2017-08-10 | 分类于 树莓派 | 阅读次数:

对于不知道拿树莓派干嘛的小白来说，给树莓派单独配显示器与键鼠显然是不合算的，而一般人手边也不会恰好有多余的显示器和键鼠。这个时候通过SSH和远程桌面用笔记本电脑控制是非常方便的。整个过程只需要一根额外的网线把树莓派和路由器连接，可以说是很经济的选择。

要下载的东西和安装的软件（Windows平台）：

  1. Raspbian镜像
    如果不熟悉Linux系统的话还是安装有桌面的版本吧。
  2. Win32DiskImager
    用来把镜像烧录到TF卡里。
  3. Advanced IP Scanner
    用来获取当前网络下各设备的IP地址，非常方便。
  4. putty
    通过SSH来访问树莓派。
