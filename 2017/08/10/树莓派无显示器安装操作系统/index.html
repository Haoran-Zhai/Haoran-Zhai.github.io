树莓派无显示器安装操作系统

发表于 2017-08-10 | 分类于 树莓派 | 阅读次数:

对于不知道拿树莓派干嘛的小白来说，给树莓派单独配显示器与键鼠显然是不合算的，而一般人手边也不会恰好有多余的显示器和键鼠。这个时候通过SSH和远程桌面用笔记本电脑控制是非常方便的。整个过程只需要一根额外的网线把树莓派和路由器连接，可以说是很经济的选择。

要下载的东西和安装的软件（Windows平台）：

  1. Raspbian镜像
    如果不熟悉Linux系统的话还是安装有桌面的版本吧。
  2. Win32DiskImager
    用来把镜像烧录到TF卡里。
  3. Advanced IP Scanner
    用来获取当前网络下各设备的IP地址，非常方便。
  4. putty
    通过SSH来访问树莓派。

烧录Raspbian

————————————————————————
支持树莓派的系统可以说是非常多，第一次安装操作系统选择Raspbian的原因也非常简单，就是易于操作而又能在网上找到很多教程。最开始的时候，因为手边有本《鸟哥的Linux私房菜》，查了一下刚好有CentOs的树莓派适配版本，所以打算安装CentOs系统。系统安装之后，折腾了好久都无法顺利安装远程桌面，不知道是什么原因。所以才转向Raspbian。最开始可以先安装Raspbian熟悉一下相关流程和Linux系统，熟练了之后再折腾别的系统吧。

把之前下载的镜像压缩包解压缩得到.img镜像，用Win32DiskImager这个软件烧录镜像到TF卡里：


选择映像文件与外接设备，直接点写入即可。
烧录完成后在TF卡的根目录下新建一个名为SSH无后缀名的空白文本文件。

获取树莓派的IP地址

————————————————————————
将烧录后的TF卡弹出，插入树莓派；用一根网线连接树莓派和路由器，接通电源。
此时树莓派已经供电启动，打开Adcanced IP Scanner，点击扫描。等待一会儿列表中将会出现所有在目前网络下的设备名称以及其IP地址，找到其中树莓派的IP地址记录下来。

通过SSH连接树莓派

————————————————————————
打开putty，在Host Name (or IP Stress)栏中输入刚刚获得的树莓派的IP地址，点击Open。


按照提示输入树莓派的用户名和密码，Raspbian默认用户名为pi，密码为raspberry。注意输入密码的时候命令行中是没有输入显示的，正确输入之后enter就可以了。


此时就可以使用命令行来控制树莓派了。

通过远程桌面连接树莓派

————————————————————————
命令行搞不懂，桌面才是目的地。远程桌面软件很多，windows附件里的远程桌面连接就非常好用。

目前只能通过SSH利用命令行来控制电脑，第一次进入系统后最好先更新一下软件：
sudo apt-get update
顺便可以设置一下密码（默认的是刚刚登陆SSH输入的raspberry）：
sudo raspi-config
