树莓派Aria2远程下载

如果没有其他什么配件的话，树莓派比较常用的一个用途就是刷一个媒体播放系统再加上远程下载做播放机用。搜了很多教程，推荐最多的就是安装Xware固件通过迅雷远程下载实现，毕竟网上资源最多而且连接稳定的还是迅雷资源。但是在尝试安装配置迅雷远程下载的过程中，发现无法获得注册码。在网上苦寻解决方案，却发现在7月17号的时候，迅雷远程下载Xware宣布关闭。也就是说网上所有的利用Xware来把树莓派打造成一个远程下载机的教程全部无效。由此只能通过Aria2实现远程下载功能了。不过感觉上Aria2的使用上比较受限，下载百度云盘里的电影倒是如鱼得水，其他种子之类的话还是迅雷好用啊（可能没有掌握正确的电影下载技巧～～）。

安装过程的话呢，推荐这篇博文–by mkitby，非常详细，亲测无误。

精简一下主要的操作步骤并具体添加一些Web页面访问和百度云下载插件的安装。

安装并配置Aria2

————————————————————————
sudo apt-get install aria2 –y
mkdir /home/pi/.aria2
touch /home/pi/.aria2/aria2.session#创建aria2配置文件
nano /home/pi/.aria2/aria2.conf#创建aria2配置文件并添加
配置文件内容如下（可以自己修改例如下载目录等项目，输入目标下载地址即可）：

sudo nano /etc/init.d/aria2c#为aria2添加自启动服务
添加如下内容，Ctrl+O后Enter保存，Ctrl+X存盘退出：

sudo chmod +x /etc/init.d/aria2c#添加可执行权限
sudo update-rc.d aria2c defaults#更新rc，添加自启动服务
sudo service aria2c restart#重启动
OK！Aria2安装并配置完成！

通过Web控制下载

————————————————————————
下载并安装Chrome上的插件YAAW进行下载控制。
点击YAAW进入Web UI界面点击右上方小扳手图标进行配置，在JSON-RPC Path栏填入http://[你的树莓派IP地址!]:6800/jsonrpc后保存。
右击YAAW的图标点击选项，在JSON-RPC的RPC Path栏依旧填入http://[依然是你的树莓派IP地址!]/jsonrpc然后保存。
安装完成之后，在下载链接处右键菜单中就能看到ARIA2 RPC选项，单击就可以添加到树莓派上进行下载。YAAW的Web UI页面中可以查看下载。对于BT下载，直接右键torrent的连接添加，下载完torrent会自动开始下载任务；当然也可以在页面手动添加下载地址和上传torrent下载。
如果想通过外网访问的话，需要路由器添加端口映射（不要使用80和8080端口）。#暂时未尝试，尝试后另附。
外网访问的话最好设置密码验证，在aria2.conf文件中添加：

在YAAW中使用http://token:secret@hostname:port/jsonrpc的地址格式设置secret.
或者：

在YAAW中使用http://username:passwd@hostname:port/jsonrpc的地址格式设置密码.

添加百度云插件

————————————————————————
下载插件BaiduExporter.

下载完成后，打开chrome插件管理,勾选开发者模式，点击“加载已解压的扩展程序”，上传刚刚下载的zip压缩包中的chrome文件夹。

打开百度云页面，出现新的“导出下载”摁钮，点击设置，同样设置树莓派的ip地址并保存。


选择需要下载的文件，“导出下载”>“ARIA2 RPC”即可添加到aria2下载，同样可以打开YAAW查看下载。
